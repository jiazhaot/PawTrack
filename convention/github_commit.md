# github commit conventions:

## basic template:

(type) [optional scope]: (description)

[optional body]

[optional footer(s)]

## brief:

- no period
- lowercase letters
- one blank space between the colon and the description
- present tense
- type:
  - feat
  - fix
  - docs
  - style
  - refactor
  - perf
  - test
  - chore
  - revert
